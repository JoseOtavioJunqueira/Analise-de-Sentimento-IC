# Como citar este projeto e os recursos utilizados

---

## Citar o projeto de IC (repositório)

Se você utilizar ou referenciar este repositório em trabalhos acadêmicos, use uma citação no formato abaixo (ajuste conforme o estilo da sua instituição — ABNT, APA, etc.):

**ABNT (exemplo):**

> RAMOS, J. O. J.; SANTELLO, L. D. Aprimoramento de Sistemas de Recomendação em Investimentos Financeiros por meio da Análise de Sentimentos. São Carlos: ICMC/USP, 2025. Repositório de Iniciação Científica. Disponível em: https://github.com/[usuario]/Analise-de-Sentimento-IC. Acesso em: dia mês ano.

**BibTeX (exemplo):**

```bibtex
@misc{ramos_santello2025_ic,
  author = {Ramos, José Otávio Junqueira and Santello, Luísa Domingues},
  title = {Aprimoramento de Sistemas de Recomendação em Investimentos Financeiros por meio da Análise de Sentimentos},
  year = {2025},
  institution = {Universidade de São Paulo, ICMC, São Carlos},
  howpublished = {\url{https://github.com/[usuario]/Analise-de-Sentimento-IC}},
  note = {Projeto de Iniciação Científica}
}
```

Substitua `[usuario]` pela URL real do repositório.

## Citar o modelo FinBERT-PT-BR

O modelo de análise de sentimentos utilizado é o **FinBERT-PT-BR**, desenvolvido por **Lucas L. Santos** na USP. A citação obrigatória é:

**Referência (ABNT):**

> SANTOS, L. L. **FinBERT-PTBR: uma abordagem baseada em BERT para análise de sentimentos de textos financeiros em português.** Dissertação (Mestrado) — Universidade de São Paulo, 2022.

**BibTeX:**

```bibtex
@mastersthesis{santos2022finbert,
  author = {Lucas L. Santos},
  title = {FinBERT-PTBR: uma abordagem baseada em BERT para análise de sentimentos de textos financeiros em português},
  school = {Universidade de São Paulo},
  year = {2022},
  type = {Dissertação de Mestrado}
}
```

**Hugging Face:**

- Modelo: [lucas-leme/FinBERT-PT-BR](https://huggingface.co/lucas-leme/FinBERT-PT-BR)

## Base de dados de notícias

A proposta do projeto menciona a utilização e ampliação da base de notícias relacionada ao trabalho de **Santos (2022)**. Ao descrever a base de dados utilizada, cite o trabalho original e, quando aplicável, este projeto como extensão/atualização da base.
